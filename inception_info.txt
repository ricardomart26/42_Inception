Comandos para fazer quando criar linux server:

apt update -y
apt upgrade -y

change timezone:
dpkg-reconfigure tzdata - set timezone


adduser <username>

Para adicionar ao sudofile:
adduser <username> sudo

Remover pass do root user:
sudo passwd -d root

Dar lock a password:
sudo passwd -l root

sudo apt install nginx

sudo mkdir /var/www/<domain or Ip addr>
sudo chmod 755 -R /var/www/<domain or Ip addr>/


Para modar o ownership da pasta para o user:
sudo chown -R <username>:www-data /var/www/<domain or Ip addr>/

Quando correr o deployment script, vai permitir atualizar o conteudo do diretorio

Criar config file
sudo nano /etc/nginx/sites-available/<domain or Ip addr>

server {
	listen 80;
	listen [::]:80;
	
	root [caminho do codigo]
	index index.html
}

Testar a config:
sudo nginx -t 

sudo unlink /etc/nginx/sites-enabled/default
sudo ln -s /etc/nginx/sites-available/<domain or Ip addr> /etc/nginx/sites-enabled/

sudo systemctl restart nginx
